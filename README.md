# FocusFlow - 番茄钟与网页使用统计 Chrome 插件

一款精致的浏览器生产力工具，帮助你专注工作并了解自己的上网习惯。

## 功能特性

### 番茄钟
- 可自定义专注时长（15/25/45/60 分钟）
- 实时倒计时显示，带环形进度条
- 开始/暂停/重置控制
- 完成后桌面通知提醒
- 今日番茄数和专注时长统计

### 网页使用统计
- 自动追踪各网站停留时间
- 每日/每周使用报告切换
- 可视化图表展示（环形图）
- 网站使用排行列表
- 专注比例计算

## 安装方法

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本插件文件夹

## 技术栈

- **Manifest V3** - Chrome 扩展最新标准
- **Service Worker** - 后台持续运行
- **Chrome Storage API** - 本地数据持久化
- **Canvas API** - 自定义图表绘制
- **CSS Variables** - 主题色彩管理

## 设计特色

- **深色工业风主题** - 深色背景配合珊瑚橙强调色
- **JetBrains Mono + Space Grotesk** - 精致的字体搭配
- **环形进度条** - 直观的倒计时可视化
- **流畅动画** - 微交互提升用户体验

## 隐私说明

- 所有数据仅存储在本地浏览器中
- 不会上传任何使用数据到服务器
- 自动清理 30 天前的历史数据

## 使用技巧

1. 点击浏览器工具栏的 FocusFlow 图标打开插件
2. 在「番茄钟」标签设置专注时长并开始计时
3. 切换到「统计」标签查看使用报告
4. 使用快捷键快速访问（需自行设置）
