# 从零开始：我是如何创建一个 Chrome 插件的
## 一个普通用户的番茄钟开发实录

---

## 写在前面

作为一个没有编程背景的普通用户，我一直想拥有一个**既能计时又能统计网页使用时间**的工具。市面上的番茄钟应用要么功能单一，要么太复杂。于是我决定——自己动手做一个！

这篇文章记录了我从零开始创建 **FocusFlow**（专注流）Chrome 插件的全过程。不需要懂代码，你只需要知道：**如果我能做到，你也可以**。

## 一、什么是 Chrome 插件？

### 简单理解

想象 Chrome 浏览器是一个房子，**插件就是你可以安装的各种家具和电器**：

- **广告拦截器** = 防盗门
- **翻译插件** = 随身翻译官
- **密码管理器** = 智能保险箱
- **FocusFlow** = 你的私人时间管家

Chrome 插件用一套标准的技术（HTML、CSS、JavaScript）写成，安装后会在浏览器工具栏显示一个小图标，点击就能使用。

### 插件能做什么？

| 功能类型 | 例子 |
|---------|------|
| 修改网页 | 改变网页颜色、字体、布局 |
| 后台运行 | 记录浏览历史、自动填充表单 |
| 弹窗界面 | 点击图标出现的操作面板 |
| 系统通知 | 定时提醒、消息推送 |

**FocusFlow 结合了后台运行 + 弹窗界面 + 系统通知三种能力。**

---

## 二、FocusFlow 是什么？

### 核心功能

FocusFlow 是一个**番茄钟 + 网页使用统计**的组合工具：

#### 1. 番茄钟功能

**什么是番茄钟？**

这是一种时间管理方法：
- 工作 25 分钟
- 休息 5 分钟
- 每 4 个番茄钟后，休息 15-30 分钟

**FocusFlow 的实现：**
- 可自定义时长（15/25/45/60 分钟）
- 环形进度条实时显示剩余时间
- 完成后桌面通知提醒
- 自动统计今日完成数量

#### 2. 网页使用统计

**它能记录什么？**

- 你在每个网站停留了多长时间
- 今天访问了哪些网站
- 专注时间 vs 浏览时间的比例

**有什么用？**

想象你本来想查资料，结果在社交媒体上刷了 2 小时——FocusFlow 会帮你发现这个问题！

## 三、开发过程揭秘

### 第一步：设计界面

**我的需求清单：**

1. 界面要好看（深色主题，不刺眼）
2. 操作简单（一键开始/暂停）
3. 数据直观（图表展示）

├─────────────────────┤
│  [番茄钟] [统计]     │  ← 标签切换
├─────────────────────┤
│                     │
│     ╭───────╮       │
│    ╱    25:00 ╲      │  ← 环形计时器
│    │  专注中... │     │

│                     │

**设计亮点：**

- **环形进度条**：比传统数字倒计时更直观

### 第二步：编写代码

**Chrome 插件的核心文件：**

| 文件 | 作用 | 类比 |
|------|------|------|
| `manifest.json` | 插件身份证 | 产品说明书 |
| `popup.html` | 点击图标显示的界面 | App 首页 |
| `popup.css` | 界面样式 | 装修方案 |
| `popup.js` | 界面交互逻辑 | 按钮功能 |
| `background.js` | 后台运行程序 | 24小时管家 |

#### manifest.json - 插件的配置文件

```json
{
  "name": "FocusFlow - 番茄钟与使用统计",
  "version": "1.0.0",
  "permissions": ["storage", "tabs", "alarms"],
  "action": {
    "default_popup": "popup.html"
  }
}
```

**关键配置解释：**
- `storage`：保存你的使用数据
- `tabs`：追踪你访问的网页
- `alarms`：定时器功能

#### background.js - 后台的大脑

这是插件最复杂的部分，它要同时处理两件事：

**1. 番茄钟计时**
```javascript
// 每秒检查一次
setInterval(() => {
  if (timerRunning && timeRemaining > 0) {
    timeRemaining--;
    updateBadge(); // 更新图标上的数字
  }
}, 1000);
```

**2. 网页追踪**
```javascript
// 监听标签页切换
chrome.tabs.onActivated.addListener(() => {
  recordTimeOnPreviousSite();  // 记录刚才网站的时间
  startTrackingNewSite();      // 开始追踪新网站
});
```

### 第三步：创建图标

**图标规格要求：**

Chrome 扩展需要 4 种尺寸的图标：
- 16×16：工具栏显示
- 32×32：Retina 屏幕
- 48×48：扩展管理页面
- 128×128：Chrome 商店

**我的设计：**

```
┌─────────────┐
│  🔶 渐变背景  │  ← 珊瑚橙渐变
│    ⭕       │  ← 白色时钟圆圈
│    │╲      │  ← 时针和分针
│    ●       │  ← 中心点
└─────────────┘
```

**设计工具：**
- 使用 SVG 格式（矢量图，任意缩放不失真）
- 用 `sips` 命令转换为 PNG

### 第四步：测试调试

**安装方法：**

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择插件文件夹

**常见问题：**

| 问题 | 解决方法 |
|------|---------|
| 图标不显示 | 检查 PNG 格式和路径 |
| 计时器不走 | 检查 background.js 是否运行 |
| 数据没保存 | 检查 storage 权限 |

## 四、功能详解

### 番茄钟界面

**操作说明：**

1. **选择时长**：点击 15/25/45/60 分钟按钮
2. **开始计时**：点击「开始」按钮，环形进度条开始走动

## 五、技术亮点

### 1. 深色/浅色模式适配

所有图标使用 `currentColor`，可以通过 CSS 控制颜色：

```css
/* 深色模式 */
.icon { color: #f0f0f5; }

/* 浅色模式 */
.icon { color: #1a1a2e; }
```

### 2. 数据本地存储

所有数据保存在浏览器本地，**不会上传到任何服务器**：

```javascript
// 保存数据
chrome.storage.local.set({
  dailyStats: {
    "2026-03-19": {
      pomodoros: 4,
      focusMinutes: 100,
      sites: { "google.com": { minutes: 30, visits: 5 } }
    }
  }
});
```

### 3. 自动清理旧数据

只保留最近 30 天的数据，避免占用过多空间。

## 六、使用建议

### 最佳实践

1. **设定每日目标**：比如「今天完成 8 个番茄钟」
2. **查看统计规律**：发现自己什么时间最容易分心
3. **识别时间黑洞**：哪些网站占用了你最多时间？
4. **逐步改善**：不要期望一天改变，每周进步一点点

### 番茄钟使用技巧

- **工作时段**：关闭手机通知，专注一件事
- **休息时段**：离开电脑，喝水、拉伸、看远处
- **记录中断**：如果被打断，记下原因，后续优化

## 附录：常见问题

**Q: 这个插件收费吗？**
A: 完全免费，开源项目。

**Q: 数据会泄露吗？**
A: 不会，所有数据只存储在你的电脑里。

**Q: 支持哪些浏览器？**
A: Chrome、Edge、Brave 等基于 Chromium 的浏览器。

**Q: 可以自定义时长吗？**
A: 可以，支持 15/25/45/60 分钟四种预设。

**Q: 如何卸载？**
A: 打开 `chrome://extensions/`，找到 FocusFlow，点击「移除」。
